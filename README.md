# Hexo-Github-Blog-Back-End
Backup for my hexo+Github blog 

## Environment
* Node.js 18.x
* Hexo 6.x+

## Usage

```shell
git clone https://github.com/ZhaoQi99/ZhaoQi99.github.io.git
git submodule init
```
## For CTF
```shell
bash ctf_post.sh new # create new post
bash ctf_post.sh replace # replace asset path
```
